# 酥橘结垢大作业

如图3-7所示，我们允许用户自行配置生成的图数据的各项参数，图示默认值为
推荐值。 
如图3-8所示，我们的可视化程序可以正确绘制节点、双向连边、坐标网格，并通
过颜色和数值标注直观地展示连边的拥堵程度。 
如图3-9所示，可视化的同时我们允许用户进行交互操作，包括： 
（1） 通过鼠标点击可视化界面的节点选择起点和终点，并选择计算距离最短路、
时间最短路等  
（2） 通过鼠标滚轮缩放界面，同时根据缩放的大小自动选择省略一些节点的绘
制（切换层级） 
（3） 查看和暂时地强制改变当前显示的层级，因为考虑省略绘制节点的时机可
能不是用户所期待的，给予用户更多的灵活性 
（4） 通过输入坐标跳转到图的特定部分 
（5） 调整模拟实时车流的仿真时间倍率，让用户能更好地观察车流数据的变化 
（6） 选择是否渲染边的数值标注和网格
